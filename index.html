How This VAT Calculator Works

This VAT calculator allows you to easily calculate the VAT amount based on the inputted value and VAT percentage. Here’s how you can use it:

  • Add VAT: If you want to find the total price including VAT, select "Add VAT" and input the base amount along with the VAT percentage.
  • Exclude VAT: If you want to find the price excluding VAT from a gross total, select "Exclude VAT" and provide the total amount and VAT percentage.

The calculator uses the following formulas:

  • Add VAT: Total = Amount + (Amount × VAT Rate)
  • Exclude VAT: Net = Amount ÷ (1 + VAT Rate)

Simply input the values and click "Calculate" to get your result!
